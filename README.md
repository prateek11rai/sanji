# Sanji

## Intro

A Personal blog site built using [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/).
The site has two sections:
- About Me : Personal Information
- Blog : Rants

## Setup

### Install Dependencies

Must have python 3.12.2 and uv installed in the system

```bash
uv sync
```

### Run the Server

```bash
uv run mkdocs serve
```

### Stop the Server

Mac: Press ^ + C
Windows/Linus: Press Ctrl + C
